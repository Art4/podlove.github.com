### Lists

> network.lists

List network lists. Use theslug parameter to access a specific list.

#### Examples

Iterate over all lists.

```twig
{% for list in network.lists %}
    {{ list.title }}
{% endfor %}
```

Access a specific list by id.

```twig
{{ network.lists({id: "example"}).title }}
```
