> Error overlay that handles issues with configuration, network issues or media issues.

### Preview

<PodloveWebPlayer style={{ marginBottom: '15px' }}
  config="/assets/web-player/v5/config.json"
>
{`
  <root class="p-4 flex justify-center h-64">
    <error></error>
  </root>
`}
</PodloveWebPlayer>

:::info

* It is only displayed if an error happens
* Network related issues will also display a retry button to recover from the error

:::
